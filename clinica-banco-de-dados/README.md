# 🏥 Sistema de Clínica - Banco de Dados

## 📌 Sobre o projeto
Este projeto simula um sistema de gerenciamento de uma clínica médica.

## 🎯 Funcionalidades
- Cadastro de médicos
- Cadastro de pacientes
- Controle de consultas
- Relacionamento entre entidades

## 🛠️ Tecnologias
- SQL (MySQL)

## 📂 Estrutura
- 01-criacao.sql → criação das tabelas
- 02-alteracoes.sql → alterações
- 03-inserts.sql → dados de teste

## ▶️ Como executar
1. Execute 01-criacao.sql
2. Execute 02-alteracoes.sql
3. Execute 03-inserts.sql

## 📚 Aprendizados
- Modelagem relacional
- Uso de chaves primárias e estrangeiras
- Organização de scripts SQL
