# Sistema de Clínica - Banco de Dados

Sistema simples com funções básicas para aprimorar e fixar conhecimentos.

## Como executar

1. Execute o arquivo 01-criacao.sql - Para criar o banco e as tabelas;
2. Execute o arquivo 02-alteracoes.sql - Alterações simples;
3. Execute o arquivo 03-inserts.sql - Adicionando dados nas tabelas;
4. Execute o arquivo 04-alteracoesFinais.sql - Excluindo e alterando dados nas tabelas.
